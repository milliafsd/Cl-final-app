# 🕌 جامعہ ملیہ اسلامیہ فیصل آباد — Smart ERP

## چلانے کا طریقہ (Streamlit Cloud)

### GitHub پر اپ لوڈ کریں:
```
app.py
requirements.txt
.streamlit/config.toml
.gitignore
```

### Streamlit Cloud پر Deploy کریں:
1. https://share.streamlit.io پر جائیں
2. New app → آپ کا GitHub repo منتخب کریں
3. Main file: `app.py`
4. Deploy!
